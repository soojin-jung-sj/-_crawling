# -_crawling
부트캠프에서 진행한 미니 프로젝트에 관한 코드 
한우 명절 선물세트 데이터 분석 프로젝트
이 프로젝트는 설 연휴를 앞두고 수요가 높은 **'한우 선물세트'**에 대한 소비자 트렌드와 리뷰 데이터를 분석하기 위해 진행되었습니다. 웹 크롤링을 통한 데이터 수집부터 전처리, 그리고 워드클라우드 및 평점 분석을 통한 인사이트 도출 과정을 포함하고 있습니다.

📂 파일 구성 및 기능
프로젝트는 데이터 처리 단계에 따라 총 3개의 Jupyter Notebook 파일로 구성되어 있습니다.

1. 명절선물세트 크롤링.ipynb
목적: 이커머스(네이버 쇼핑 등) 플랫폼에서 한우 선물세트 관련 데이터를 자동으로 수집합니다.

주요 기능:

Selenium과 chromedriver_autoinstaller를 활용한 웹 브라우저 제어.

상품명, 가격, 별점, 구매 날짜, 리뷰 텍스트 데이터 추출.

tqdm 라이브러리를 사용한 크롤링 진행률 시각화.

2. 프로젝트 데이터 정리.ipynb
목적: 수집된 원본 데이터(Raw Data)를 분석에 적합한 형태로 정제합니다.

주요 기능:

중복 데이터 제거 및 결측치 처리.

분석에 필요한 주요 컬럼(주문날짜, 별점, 상품명, 후기 등) 선별.

최종 분석용 데이터셋(한우세트_최종데이터.csv) 생성.

3. 한우_데이터_분석해보기.ipynb
목적: 정제된 데이터를 시각화하고 소비자 반응을 분석합니다.

주요 기능:

워드클라우드(Word Cloud): 리뷰 텍스트를 분석하여 '맛', '배송', '신선도' 등 주요 키워드 시각화.

배송 만족도 분석: '배송', '도착' 등 특정 키워드가 포함된 리뷰를 필터링하여 물류 서비스 만족도 확인.

트렌드 분석: 날짜별 주문수 분포 및 상품별 평균 별점 비교 분석.

🛠 기술 스택
Language: Python 3.x

Library:

Data Analysis: Pandas, NumPy

Web Scraping: Selenium, BeautifulSoup, chromedriver_autoinstaller

Visualization: Matplotlib, WordCloud

Utilities: tqdm, time, warnings

📈 분석 결과 요약
소비자 만족도: 대부분의 상품이 4.9 이상의 높은 평점을 유지하고 있으며, 특히 선물용으로서의 만족도가 높음.

핵심 키워드: '신선함', '빠른 배송', '재구매' 등의 단어가 빈번하게 등장함.

배송 서비스: 명절 전 특정 시점에 주문이 집중되나, 전반적으로 배송 날짜 준수 및 포장 상태에 대해 긍정적인 평가가 많음.

본 프로젝트는 데이터 수집부터 시각화까지의 데이터 분석 파이프라인을 학습하기 위한 목적으로 제작되었습니다.
